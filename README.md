# Proyecto Camarón
## Manual de Usuario
> Fecha de realización **08/Dic/2025**

En el archivo de GitHub viene una descarga con una base de datos, esta permite al usuario abrir la base de datos desde su servidor Apache MySQL. El usuario deberá de iniciar los módulos de Apache y MySQL en el panel de contról de XAMPP y dar click en el botón de Admin de MySQL.

Una vez adentro se creará una nueva base de datos con el nombre del archivo con terminación .bd (`cproject`). En esta es en la que se va a importar la base de datos usando el botón de importar y luego subiendo el archivo.

Después de subir el archivo ya se puede usar la página, solo falta ponerla en el lugar correcto. Al descargar XAMPP se crea una carpeta especial en la que se pueden visualizar páginas web, esta tiene normalmente como ruta `C:\xampp\htdocs`. Adentro de esta se meterá toda la carpeta de "Proyecto-Camaron".

Para simplicidad de uso, no se tiene que poner `localhost/php/Proyecto-Camaron/homepage/homepage.html`. Esto es tardado, largo y complejo, debido a esto, para entrar a la página solo se tiene que escribir el siguiente link. `localhost/Proyecto-camaron`. 

Después de entrar a la página principal, el usuario puede entrar a la página de login por medio del botón de arriba a la derecha con el nombre de "Log In". De aquí se puede iniciar sesión cuando ya haya una cuenta creada. 

Para crear una cuenta se tiene que dar click en el botón de abajo a la izquierda que dice "Regístrate",  Este proceso pide 6 datos diferentes, todos son obligatorios menos el nombre y la foto de perfil, en caso de no poner nombre se usará el nombre de usuario y en caso de no poner foto de perfil se usará una por defecto.

Después de crear la cuenta se puede regresar a la página principal e iniciar sesión. Al iniciar sesión se entrará a la página principal, en esta saldrán todas las lecciones disponibles para que el usuario pueda elegir una y leer toda.

Si se decide cambiar la configuración el usuario puede dar al botón de "Configuración" desde el menú y hacer cambios de su foto de perfil, usuario, contraseña y/o correo. Al terminar se da click en guardar cambios.

La ventana de créditos tiene un link al proyecto de GitHub junto con los nombres de todo el equipo de desarrollo
